# 🌐 Umair Ali — Personal Portfolio Website

A modern and responsive personal portfolio website built with pure HTML, CSS and JavaScript, showcasing my skills, projects, and journey as a front-end web developer. Designed to make a strong first impression with a clean, professional layout.

---

## 🔗 Live Demo

🌍 https://umairalimalik78.github.io/portfolio/

---

## ✨ Features

- 👋 Hero / Introduction section with name and tagline
- 👤 About Me section with personal summary
- 🛠️ Skills section showcasing technologies known
- 💼 Projects section with project cards and links
- 📬 Contact section with email or form
- 🌙 Clean and modern UI design
- 📱 Fully responsive layout for all screen sizes
- 🌐 Built with pure HTML & CSS — no frameworks

---

## 🚀 Getting Started

### Run Locally

1. **Clone the repository**

```bash
git clone https://github.com/Umairalimalik78/New-Portfolio.git
```

2. **Navigate to the project folder**

```bash
cd New-Portfolio
```

3. **Open `index.html` in your browser**

Simply double-click the `index.html` file or open it with **Live Server** in VS Code.

> No build tools, frameworks, or installations required — it's pure HTML & CSS!
